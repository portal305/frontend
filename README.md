# Ember

![App Icon](icon.png)


[![Discord](https://img.shields.io/badge/discord-5865F2?logo=discord&logoColor=white)](https://discord.gg/8CCNRpj2kg)

**Ember** is a modern, feature-rich frontend emulator designed to provide an elegant and seamless retro gaming experience. Built with Flutter, it serves as a sophisticated launcher and library manager for your retro game collection, offering a visually stunning interface and powerful organization tools.

> **Note:** This is a **closed-source** application. The source code is not available for public distribution or modification.

*   **Multi-Layout Home Screen:** Choose from various dynamic layouts to suit your style:
    *   **Carousel:** A classic, cover-flow style view.
    *   **Dashboard:** An information-rich overview of your systems and recent activities.
    *   **Flow:** A smooth, continuous browsing experience.
*   **Comprehensive Library Management:** Organize your game collections with ease. Ember supports detailed game metadata, box art, and efficient filtering.
*   **News & Updates:** Stay informed with a dedicated News section featuring:
    *   **App News:** The latest updates and changelogs for Ember.

*   **Game Details:** Deep dive into your games with rich detail screens displaying descriptions, release dates, and related media.
*   **Profile Integration:** Manage your user profile and track your gaming stats.
*   **Theming:** Full support for Light and Dark themes to match your preference or environment.
